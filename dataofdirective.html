Now we are learn about Angularjs directives

Follwing are the list of angularjs directives

    Normal Directives.

  1. ng-app: It indicates the start of an angularjs apllication.
  2. ng-model:It binds the value of input html controlles/html tag to
    apllication data.If you see this cocept from programming point
    of view then it is nothimg but a variable declartion.
  3. ng-init: It use to initiate the varible value
  4. ng-repeate:Its just like an loop.
  5. ng-hide:It is used to hide or show the some information.
  6. ng-show:It is used to hide or show the some information.
  7. ng-disable:Its used to disable or enable the content.
  8. ng-enable:Its used to disable or enable the content.
  9. ng-controller:It is used to define the controller in the
    application
  10. ng-model:It is used to define the model in application

    Form controller Directives.

  • ng-required:It is use to from validation,
    for making the any form filed compulsary.
  • ng-pristine:it returns true if the form is not uodated or touched
  • ng-dirty:it returns true if the form is uodated or touched
  • ng-valid:It returns true if the field is valid
  • ng-invalid:It returns true if the field is invalid
  • ng-copy:Its use to returm if the some content is copied.
  • ng-paste:Its use to returm if the some content is pasted.
  • ng-minlength:It defines the minimum length for form filed.
  • ng-maxlength:It defines the maximum length for form filed.
  • ng-touched:it returns true if the form filed is used.
