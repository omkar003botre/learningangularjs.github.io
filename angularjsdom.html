AngularJS HTML DOM

AngularJS has directives for binding application data to attributes
of the HTML DOM elements.

The ng-disabled Directive

The ng-disabled directive binds AngularJS application data to the disabled
attribute of HTML elements.

Click the button to disable the Button


The value is {{dis}}

Explination

Application explained:

The ng-disabled directive binds the application data mySwitch to the
HTML button's disabled attribute.
The ng-model directive binds the value of the HTML checkbox
element to the value of mySwitch.
If the value of mySwitch evaluates to true, the button will be disabled:
