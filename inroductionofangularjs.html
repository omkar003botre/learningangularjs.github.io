What is angular js?

-> Angular js is the javascript framework,It can be added to the html by using the script tag.
-> Angular js is extends html element or attribute by using directives

How the angularjs extends the html?

Angularjs extends the html by using the angularjs directives.

list of the directives used in angular js

  1. ng-app:This direct defines angularjs application.
    In other word it tells to the angularjs that the tag in which ng-app is
    used is the initial of that applcation or the start of the angular js.
  2. ng-model:This directive binds the value of html
    controls like(input,select tag) to the application data.
  3. ng-binds:This directive binds the application data to the html
    view.Its just opposite to the ng-model.
  4. ng-init:This directive initiate the value of angularjs application.
